# Bugfix

1. Reproduce or explain the suspected cause.
2. Locate the smallest responsible area.
3. Avoid unrelated refactors.
4. Add validation or tests when practical.
5. Preserve existing GraphQL contracts unless the bug is the contract.
6. Check auth, ownership and Flutter impact.
7. Summarize root cause and validation.
